log₂ で巻いた周波数軸(螺旋、A1 55Hz → A7 3520Hz)。中心への射影 = ピッチクラス円。同じ音名は同じ光線に並ぶ
巻く前の姿 — 直線のログ周波数軸。×2 ごとに等間隔。これをひと巻き=1オクターブで巻いたのが上の螺旋
周波数 440.0 Hz 音名 A4 円上の位置 900¢

440Hzのラと880Hzのラは、高さは違うが「どちらもラ」だ — 耳と目で確かめる。 ラを何個か鳴らすと、螺旋上の高さは違うのに全部同じ光線に落ちる。 五度上(660Hz)や長三度上(550Hz)は別の光線だ。単純な比は溶け合い、複雑な比はザラつく — スライダーで円を回しながら440に重ねると、協和の谷と不協和の山を連続で体感できる。 円周の12目盛りは現代の地図として薄く敷いてある(本文§2以降の産物)。

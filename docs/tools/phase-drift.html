ピッチとリズムは、~20Hz で割れた同じ周波数軸。 3:2 は速度側では 3対2 のポリリズム、ピッチ側では 完全五度。 タイムラインで A と B がズレて 最小公倍数でぴたりと揃い直すのは、単発の純正比が 閉じる＝協和の姿（コンマではない）。コンマは別物 — この比を積んだときだけ現れ、右パネルのオーバーシュートが示す。 テンポを可聴域（20Hz 超）へ押し上げれば打点は音程に溶ける — その先は octave-circle（ピッチの円）と polyrhythm-player（時間のモード）へ。
数値は fabrication ではなく (3/2)^12 / 2^7 等から計算 — ピタゴラスコンマ 23.460¢ = 1.36%。
